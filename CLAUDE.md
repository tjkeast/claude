# CLAUDE.md

## Standing task: Daily healthcare-payments research

There is a recurring daily research project (target: **10:00 AM daily**) into the
**healthcare payments industry**, starting from **HICAPS** and broadening out.

When a session is for this task, follow the runbook in
[`research/healthcare-payments/README.md`](research/healthcare-payments/README.md):

1. Read `research/healthcare-payments/research-log.md` (everything covered so far).
2. Read `research/healthcare-payments/topics-backlog.md` and pick the next angle(s)
   — **do not repeat** prior coverage unless there's a material update.
3. Research (web search + primary sources).
4. Append a new dated entry at the top of the entries in `research-log.md`.
5. Update `topics-backlog.md` and `entities.md`.
6. Commit and push to the working branch.

The committed files in `research/healthcare-payments/` are the project's memory —
each session runs in a fresh container, so nothing persists unless committed.
