# Healthcare Payments Industry — Daily Research

A standing research project tracking the healthcare payments industry, starting
with **HICAPS** and broadening out to the wider Australian and global landscape.

## Cadence
- **Frequency:** Daily, 10:00 AM (user's local time)
- **Started:** 2026-06-07
- **Owner:** keast.taylar@gmail.com

## How this works (read before each run)
This repo is the **memory**. Each daily session is a fresh container, so the only
thing that persists is what is committed here. Every run should:

1. Read [`research-log.md`](./research-log.md) to see everything covered so far.
2. Read [`topics-backlog.md`](./topics-backlog.md) to pick the next angle(s) — do
   **not** repeat what's already been covered unless there's a material update.
3. Do the day's research (web search + fetch primary sources where possible).
4. Append a new dated entry to `research-log.md` (newest at top of the entries).
5. Update `topics-backlog.md` (tick off what was done, add new threads found).
6. Update `entities.md` if new companies/products/people are worth tracking.
7. Commit and push to the working branch.

## Files
- `research-log.md` — dated installments, the running record of findings.
- `topics-backlog.md` — queue of angles to research, so days don't overlap.
- `entities.md` — key companies, products, and people in the space.

## Scope
Primary focus: payment & claiming rails for healthcare (private health insurance
claiming, Medicare claiming, EFTPOS at point of care, patient billing). Geography
starts Australia-centric (HICAPS) and expands to global comparisons and trends.
